Does Biological Connectome Topology Change Neural Network Behavior?

🇬🇧 Paper 🇫🇷 Papier Code
Abstract

The question, precisely stated

We study whether the connectivity topology of a real biological connectome, when imposed as a fixed binary mask on an artificial neural network, affects learning dynamics, generalization, and robustness. The C. elegans hermaphrodite connectome (448 neurons, 4,681 synapses, OpenWorm) provides a full 448×448 adjacency matrix, binarized and applied as a non-trainable mask to both feedforward and recurrent networks.


We benchmark against three controls across four tasks, 5 random seeds, and 100 training epochs. Bio-topological connectivity performs equivalently to density-matched random sparsity on all tasks. Dense networks retain a consistent advantage on regression tasks only. The full 448×448 matrix must be used unchanged to preserve the global properties (modularity Q = 0.446, 35 communities, clustering coefficient 0.295) that motivate the hypothesis.

TL;DR — Biological wiring neither hurts nor helps: bio-topological masks perform identically to random sparse masks of the same density (2.33%). Stage 1 establishes the methodology; Stages 2–5 will explore richer biological priors.
Approach

How the mask works

The C. elegans chemical synapse adjacency matrix is binarized and registered as a non-trainable buffer inside each layer. During training, weights are updated by gradient descent, but those outside the mask are zeroed at every forward pass.

W_eff = W ⊙ M M ∈ {0, 1}^(448×448), fixed — never updated

Gradients at masked positions are exactly zero. The mask is a structural constraint — topology is fixed before training starts and remains unchanged throughout. The default hidden size is 448, matching the connectome exactly so the full matrix is used without any crop or tiling.

Run the benchmark

bash 
# Install
pip install -e .

# Full benchmark: 4 conditions × 4 tasks × 5 seeds
python experiments/run_celegans.py --task all --epochs 100 --output results/
python 
import wiringmatters as wm

G, adj = wm.load_celegans(synapse_type="chemical", as_matrix=True)
mask   = wm.bio_mask(adj, target_shape=448)  # full 448×448

model  = wm.BioMLP.from_connectome(
    adj, layer_sizes=[784, 448, 448, 10], mask_type="bio"
)
Roadmap

Five-stage research program

STAGE 01
Topology as Fixed Binary Mask
Gradient-trained networks with C. elegans connectome mask. Benchmark against dense, sparse, and magnitude-pruned baselines across four tasks.
Current
STAGE 02
